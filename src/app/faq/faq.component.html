FAQ

Cette application est un Système de Gestion des Ressources Humaines (SGRH) conçu pour gérer les données des employés, gérer les demandes de congés, et plus encore.

Vous pouvez demander un congé via la section "Demander un congé" dans la barre latérale.

Vous pouvez mettre à jour vos informations de profil en accédant à "Mon Profil" dans le menu déroulant.

Vous pouvez consulter les politiques de l'entreprise dans la section "Ressources" de l'application.
